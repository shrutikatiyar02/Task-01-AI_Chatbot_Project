# AI Chatbot

## Project Description
AI Chatbot is a Python-based conversational assistant that interacts with users through text commands. It can answer basic questions, provide the current date and time, and display motivational quotes.

## Features
- Greeting responses
- Date and time information
- Motivational quotes
- AI-related information
- Chat history saving
- Help menu

## Technologies Used
- Python
- datetime module
- random module

## How to Run
1. Clone the repository
2. Open terminal
3. Run:
   python chatbot.py
